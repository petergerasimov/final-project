Hello, If you want to compile the project you need to go to https://github.com/petergerasimov/physx5-for-unity (it contains some fixes on top of the original physx5-for-unity) and follow the install instructions, then replace the path of this addon with your local path.
I tried making it as a Unity package from git, but the binaries are just too large, and GitHub doesn't allow it.

There is a compiled version in the Releases tab if you want to try the game, but you need an NVIDIA graphics card.
